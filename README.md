## Description

NFT collection implemented by 
[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Run via docker

```bash
$ make build
```

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
